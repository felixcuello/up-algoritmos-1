# Módulo 3: Notación Asintótica

## Temas cubiertos
- Notación asintótica (O, Ω, Θ)
- Problemas P y NP
- Problemas computables y no computables
- Notación asintótica condicional
- Operaciones con notaciones asintóticas

## Ejercicios (Easy → Medium → Hard)

### 1. Binary Search
- **Dificultad**: Easy
- **Link**: [LeetCode #704](https://leetcode.com/problems/binary-search/)
- **Conceptos**: Binary search, análisis O(log n), comparación con búsqueda lineal

### 2. Search Insert Position
- **Dificultad**: Easy
- **Link**: [LeetCode #35](https://leetcode.com/problems/search-insert-position/)
- **Conceptos**: Binary search, notación O(log n) vs O(n)

### 3. First Bad Version
- **Dificultad**: Easy
- **Link**: [LeetCode #278](https://leetcode.com/problems/first-bad-version/)
- **Conceptos**: Binary search, análisis de complejidad logarítmica

### 4. Valid Perfect Square
- **Dificultad**: Easy
- **Link**: [LeetCode #367](https://leetcode.com/problems/valid-perfect-square/)
- **Conceptos**: Binary search, análisis O(log n) sin usar sqrt

### 5. Intersection of Two Arrays
- **Dificultad**: Easy
- **Link**: [LeetCode #349](https://leetcode.com/problems/intersection-of-two-arrays/)
- **Conceptos**: Hash tables, análisis O(n+m) vs O(n log n)

### 6. Missing Number
- **Dificultad**: Easy
- **Link**: [LeetCode #268](https://leetcode.com/problems/missing-number/)
- **Conceptos**: Math, comparación de complejidades O(n log n), O(n)

### 7. Sqrt(x)
- **Dificultad**: Easy
- **Link**: [LeetCode #69](https://leetcode.com/problems/sqrtx/)
- **Conceptos**: Binary search, análisis O(log n)

### 8. Find Peak Element
- **Dificultad**: Medium
- **Link**: [LeetCode #162](https://leetcode.com/problems/find-peak-element/)
- **Conceptos**: Binary search, análisis O(log n) vs O(n)

### 9. Search in Rotated Sorted Array
- **Dificultad**: Medium
- **Link**: [LeetCode #33](https://leetcode.com/problems/search-in-rotated-sorted-array/)
- **Conceptos**: Binary search modificado, mantener O(log n)

### 10. Find Minimum in Rotated Sorted Array
- **Dificultad**: Medium
- **Link**: [LeetCode #153](https://leetcode.com/problems/find-minimum-in-rotated-sorted-array/)
- **Conceptos**: Binary search, análisis de casos, complejidad logarítmica

### 11. Kth Smallest Element in a BST
- **Dificultad**: Medium
- **Link**: [LeetCode #230](https://leetcode.com/problems/kth-smallest-element-in-a-bst/)
- **Conceptos**: Trees, análisis O(h + k) vs O(n)

### 12. Search a 2D Matrix II
- **Dificultad**: Medium
- **Link**: [LeetCode #240](https://leetcode.com/problems/search-a-2d-matrix-ii/)
- **Conceptos**: Binary search 2D, análisis O(m + n) vs O(m log n)

### 13. Find First and Last Position of Element in Sorted Array
- **Dificultad**: Medium
- **Link**: [LeetCode #34](https://leetcode.com/problems/find-first-and-last-position-of-element-in-sorted-array/)
- **Conceptos**: Binary search doble, análisis O(log n) estricto

### 14. Kth Largest Element in an Array
- **Dificultad**: Medium
- **Link**: [LeetCode #215](https://leetcode.com/problems/kth-largest-element-in-an-array/)
- **Conceptos**: QuickSelect, análisis O(n) promedio vs O(n log n)

### 15. Find K Closest Elements
- **Dificultad**: Medium
- **Link**: [LeetCode #658](https://leetcode.com/problems/find-k-closest-elements/)
- **Conceptos**: Binary search, análisis O(log n + k)

### 16. Capacity To Ship Packages Within D Days
- **Dificultad**: Medium
- **Link**: [LeetCode #1011](https://leetcode.com/problems/capacity-to-ship-packages-within-d-days/)

### 17. Split Array Largest Sum
- **Dificultad**: Hard
- **Link**: [LeetCode #410](https://leetcode.com/problems/split-array-largest-sum/)
- **Conceptos**: Binary search vs DP, análisis O(n log(sum)) vs O(n²m)

### 18. Median of Two Sorted Arrays
- **Dificultad**: Hard
- **Link**: [LeetCode #4](https://leetcode.com/problems/median-of-two-sorted-arrays/)
- **Conceptos**: Binary search, análisis estricto O(log(min(m,n)))

### 19. Find K-th Smallest Pair Distance
- **Dificultad**: Hard
- **Link**: [LeetCode #719](https://leetcode.com/problems/find-k-th-smallest-pair-distance/)
- **Conceptos**: Binary search + two pointers, análisis O(n log n + n log(max))

### 20. Count of Range Sum
- **Dificultad**: Hard
- **Link**: [LeetCode #327](https://leetcode.com/problems/count-of-range-sum/)
- **Conceptos**: Divide and conquer, análisis O(n log n), problema difícil
